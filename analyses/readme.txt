busco.scripter.R: The script that runs BUSCO for all of the genomes

calc.tiprates.R: The script that calculates tip rates for each of the taxa

cent.vs.cont.R: This script runs ANOVA and calculates p-values for 2-6mers, totalmers using centromere type and micro satellite content

chrom.vs.content.R: This script runs ANOVA and calculates p-values for 2-6mers, totalmers using chromosome number and microsatellite content

chrom.vs.rate.R: This script runs ANOVA and calculates p-values for 2-6mers, totalmers using chromosome number and rate of microsatellite evolution

gsz.vs.content.R: This script runs ANOVA and calculates p-values for 2-6mers, totalmers using genome size and microsatellite content

gsz.vs.rate.R: This script runs ANOVA and calculates p-values for 2-6mers, totalmers using genome size and rate of microsatellite evolution

holo.mono.rate.species.R: The script that gathers Brownian motion data for centromere type across our phylogeny

microsat.chrom.R: The script that writes a data file that contains micro satellite data, genome size and diploid chromosome number. 

microsatcontent.holo.mono.R: The scrip that runs looks at micro satellite content and holocentric and monocentric chromosomes and runs aovphylo. 

order.content.R: The scrip that calculates the micro satellite content across each order

order.rates.R: The script that calculates the rate of micro satellite evolution across each order

process.BUSCO.R: The script that processes all of the BUSCO results for all of the genomes
